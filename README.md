# Cedar Sage 

A modern, responsive restaurant frontend built with **React** and **Vite**. Cedar Sage showcases a clean user interface, reusable component architecture, and a scalable structure ready for backend integration. Perfect for showcasing frontend development skills in React, responsive design, and component-based architecture.

## Features

### Core Pages
- ** Home** - Engaging landing page with hero section and highlights
- **Restaurant** - Overview of the restaurant ambiance and offerings
- **Reservations** - Interactive booking system UI
- **Cooking Classes** - Section for culinary workshops and events
- **Blog** - Articles, recipes, and restaurant updates
- **Menu** - Display of food and beverage offerings
- **Contact** - Contact information and inquiry form

### Technical Highlights
- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Component-based architecture for reusability
- ✅ Modular navigation with dynamic routing
- ✅ Optimized CSS with custom styling
- ✅ Fast development server with Vite
- ✅ ESLint configured for code quality
- ✅ Production-ready build configuration
- ✅ Scalable folder structure for future backend integration

## Getting Started

### Prerequisites

Ensure you have the following installed:
- **Node.js** (v16 or higher)
- **npm** or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ishika1106/Cedar-Sage.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Cedar-Sage
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open in your browser**
   ```
   http://localhost:5173/
   ```

## Design Philosophy

Cedar Sage follows modern frontend development best practices:

- **Component Reusability**: Common UI elements are abstracted into reusable components
- **Responsive First**: Mobile-first approach ensuring compatibility across all devices
- **Clean Code**: Organized file structure with meaningful naming conventions
- **Performance**: Optimized bundle size and fast load times with Vite
- **Maintainability**: Modular architecture for easy updates and feature additions
